简介

制定本规范的目的在于使我们的html代码更加易于维护和重用~！鉴于历史原因，本规范主要针对新项目~！当然，旧项目也可以借鉴~！

文件规范

  • 新页面的DTD声明统一为HTML5版本声明~！但html编码要求，还是严格遵循(x)html的风格；
  • 文件格式统一使用gb2312编码；
  • 禁止使用<meta http-equiv="X-UA-Compatible" content="IE=7" />来限制IE版本；
