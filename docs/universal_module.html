简介

经常用到的模块，我们通常希望能够复用！这样，在保证代码的一致性时，又能提高效率~！

分页模块

  • 医院营销中心分页的通用结构：

    说明：span表示不可点击；em表示当前页；a表示可点击；

  • 设计与制作时，应当考虑分页模块在列表页面、终端页面的通用性；
  • 制作时，考虑页码的点击区域为100%；

面包屑模块

  • 医院营销中心面包屑的通用结构：

  • 在默认情况下，该模块的设计应当以简洁为主；
  • 在默认情况下，面包屑不应该出现在首页；
  • 当某个页面出现面包屑模块的时候，h1标签应当仅出现在列表页当中；（页面有元素来表现该网页主题的时候(非面包屑模块)，h1标签在这里处理；页面无元素是表现该网页主题的时候，h1放在面包屑模块！）

文章上、下篇导航

  • 医院营销中心文章上、下篇导航的通用结构：

  • 默认情况下，建议将文字直接居中显示；
  • 可使用:first-child和:last-child为高级浏览器作特殊效果，比如： 
    .artNav li:first-child { text-align:left; /* 让上一篇导航的文字向左对齐 */ }
.artNav li:last-child { text-align:right; /* 让下一篇导航的文字向右对齐 */ }

页面尾部模块

  • 医院营销中心页面尾部通用结构

  • 默认情况下，请考虑尾部的通用性；
  • 如果项目使用CMS，尤其是医院营销中心的CMS，不建议将《友情链接》这样的模块写到公共尾部当中；
