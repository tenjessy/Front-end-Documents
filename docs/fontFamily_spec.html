简介

衬线字体：serif

比如这里的英文：My name is JunyongChen和这里的《这里讨论的是衬线字体》；

总结：只要满足末端加强原则的字体都是衬线字体。所谓的末端加强，就是使用衬线或粗细变化，使字体笔画的末端得到加强，以改善小号文字的可读性。

非衬线字体：sans-serif

比如这里的英文：My name is JunyongChen和这里的《这里讨论的是非衬线字体》；

总结：衬线字体以外的一切字体都是无衬线字体，无衬线字体比较圆滑，线条一般粗细均匀。比较适合用作艺术字、标题等。因为无衬线字体通常粗细比较均匀，所以在小字体显示的时候，可读性会降低，容易引起视觉疲劳。

目前医院营销中心统一的web字体方案：

font-family:"\5B8B\4F53", Arial, Tahoma, Helvetica, sans-serif;

注意事项

  1. 以上写法只作为参考，真实项目当中一般在reset当中对字体进行统一；
  2. 细节地方可单独书写字体，如一级标题：font-family:"\9ED1\4F53"; font-size:16px; font-weight:700; line-height:28px; （简写成：font:700 16px/1.75 "\9ED1\4F53";）
  3. 鉴于目前上线的站点均采用宋体（"\5B8B\4F53"）作为首选，因此在确定此规范后，字体依然是采用宋体作为首先字体；
  4. 样式表当中的中文字字体写法，统一使用 ascii 字符表示，使得在任何编码下都无问题；详情可参考《中文字体拼音及ascii对照表；》
  5. 默认情况下，window xp之前的操作没有开启字体平滑效果，因此在使用微软雅黑等非衬线字体的时候应该要充分考虑清楚！
